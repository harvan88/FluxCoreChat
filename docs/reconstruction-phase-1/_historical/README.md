# Documentación histórica reubicada

Esta carpeta contiene markdowns históricos reubicados para que no compitan con la documentación activa.

## Regla principal

La documentación activa del sistema está en:

- `docs/reconstruction-phase-1`

## Qué hay aquí

- `chatcore/`
- `fluxcore/`
- `kernel/`
- `archive/`
- `refactors/`
- `root-docs/`
- `repo-root-docs/`

## Cómo debe usarse

- como insumo histórico
- como evidencia de auditoría o evolución
- como fuente conceptual solo si una definición se revalida contra el código actual

## Cómo no debe usarse

- como fuente activa de verdad
- como referencia automática para describir el sistema presente
- como lugar para seguir escribiendo arquitectura nueva
